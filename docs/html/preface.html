Preface

This guide describes how to install, configure, and run the Technology Compatibility Kit (TCK) that is used to test the Jakarta Mail API (Jakarta Mail 2.0) technology.

The Jakarta Mail TCK is a portable, configurable automated test suite for verifying the compatibility of a vendor’s implementation of the Jakarta Mail 2.0 Specification (hereafter referred to as the vendor implementation or VI). The Jakarta Mail TCK uses the JavaTest harness version 5.0 to run the test suite

If you have questions about this Specification you may send inquiries to mail-dev@eclipse.org. If you have questions about this TCK, you may send inquiries to mail-dev@eclipse.org.

Who Should Use This Book

This guide is for vendors that implement the Jakarta Mail 2.0 technology to assist them in running the test suite that verifies compatibility of their implementation of the Jakarta Mail 2.0 Specification.

Before You Read This Book

You should be familiar with the Jakarta Mail 2.0, version 2.0 Specification, which can be found at https://jakarta.ee/specifications/mail/.

Before running the tests in the Jakarta Mail TCK, you should familiarize yourself with the JavaTest documentation which can be accessed at the JT Harness web site.
